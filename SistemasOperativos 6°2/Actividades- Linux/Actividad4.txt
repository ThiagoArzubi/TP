Preguntas

1. ¿Qué comando se utiliza para crear un nuevo directorio en Linux?
a) touch
b) mkdir
c) rm
d) ls

2. ¿Qué comando permite ver el contenido de un directorio?
a) cd
b) pwd
c) ls
d) cat

3. ¿Cuál es la función del comando cd en el sistema Linux?
a) Copiar archivos
b) Cambiar de directorio
c) Ver el contenido del archivo
d) Eliminar un archivo

4. ¿Qué comando se utiliza para borrar un archivo?
a) rm
b) rmdir
c) delete
d) erase

5. ¿Qué diferencia existe entre rm y rmdir?
a) Ninguna, ambos borran archivos
b) rm borra archivos, rmdir borra directorios vacíos
c) rm borra todo tipo de datos, rmdir los mueve
d) rm solo borra carpetas, rmdir archivos

6. ¿Cuál de los siguientes comandos permite copiar un archivo?
a) mv
b) cp
c) copy
d) scp

7. ¿Qué comando se emplea para mover o renombrar archivos?
a) rename
b) mv
c) move
d) cp

8. ¿Qué información muestra el comando pwd?
a) El tamaño del directorio actual
b) Los archivos ocultos
c) La ruta del directorio actual
d) La contraseña del usuario

9. ¿Qué significa el uso del parámetro -r en el comando rm?
a) Muestra los archivos eliminados
b) Elimina solo archivos con permisos
c) Elimina directorios de forma recursiva
d) Requiere confirmación antes de eliminar

10. ¿Qué hace el comando touch?
a) Crea un archivo vacío o actualiza su fecha
b) Abre un archivo
c) Muestra el contenido de un archivo
d) Copia el archivo a otro directorio

---

Respuestas

1. b) mkdir


2. c) ls


3. b) Cambiar de directorio


4. a) rm


5. b) rm borra archivos, rmdir borra directorios vacíos


6. b) cp


7. b) mv


8. c) La ruta del directorio actual


9. c) Elimina directorios de forma recursiva


10. a) Crea un archivo vacío o actualiza su fecha
